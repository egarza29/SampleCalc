# SampleCalc OpenSpec - Overview

## Vision
Deliver a fast, accessible, multi-platform scientific calculator with a robust expression engine and consistent UX across Windows, iOS, macOS, and Android.

## Scope
- Core: Expression parsing (Tokenizer → RPN → Evaluator), scientific functions, angle modes, memory, history.
- UI: Touch-friendly keypad, theme, accessibility, i18n.
- Packaging: Store-ready for Microsoft Store, App Store (iOS/macOS), Google Play.

## Non-Goals (Phase 1)
- Graphing, programming mode, complex numbers.
- Advanced CAS features.

## Success Metrics
- >95% unit test coverage on core engine.
- Store compliance with zero blocking issues.
- Cold start < 200ms on mid-range devices.

## Publisher
- OldDogSoft LLC is the organization responsible for development and store publishing.
