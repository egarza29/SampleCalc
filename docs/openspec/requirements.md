# Requirements

## Functional
- Evaluate arithmetic and scientific functions: + - * / ^ % ! sin cos tan asin acos atan ln log sqrt abs min max round floor ceil pow.
- Angle modes: degrees and radians.
- Constants: pi, e.
- Memory operations: M+, M-, MR, MC.
- History of expressions and results.

## Non-Functional
- Responsive UI; keyboard and touch input.
- Error handling: syntax, domain errors, divide-by-zero.
- Multilingual UI: English and Spanish in Phase 1, with device language default and manual override.
- Linting and static analysis clean (flutter analyze).

## Constraints
- Single codebase across mobile and desktop.
- No backend required; all offline.

## Publishing
- Publisher/developer name for stores: OldDogSoft LLC.
